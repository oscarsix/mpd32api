Mpd32Api
========

Control akai mpd32

**Features**
* Select input / output midi device with cli
* Read from midi input
* Select control bank
* Select pad bank
* Toggle note repeat
* Set note repeat value


## Installation

Add this line to your application's Gemfile:

    $ gem 'mpd32api'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mpd32api

## Usage

See bin/mpd32api
